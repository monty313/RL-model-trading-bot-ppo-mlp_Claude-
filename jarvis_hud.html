◆ J.A.R.V.I.S. — QUICK START

    
A live cockpit for the RL trading bot. It runs on its own — no setup needed.

    

      
WHAT THIS ISCenter = the data pipeline (each node is a module). Right = FTMO vitals
        (reward, win-rate, daily P&L vs +2.5%, drawdown vs the −4% wall). Bottom = the telemetry log.

      
SIM vs LIVE · top-right pillSIM = built-in demo data (default,
        so it's never blank). LIVE = real training telemetry is streaming in.

      
HOW TO GO LIVEIn a terminal, in the project folder:

        pip install websockets
python -m barbershop.ws_broadcaster --demo

        then refresh this page → the pill turns green and nodes pulse with the stream.

      
EXPLOREDrag the canvas to pan · scroll to zoom · drag a node to move it · click a node
        to read its function and open its code on GitHub · ▶ TRACE A TRADE walks one trade from
        data-in all the way to a CLOSE.

      
THE OTHER TOOL — the dashboardFor the 6 diagnostic screens (Scoreboard, Trade Autopsy,
        Repo Map, Risk Doctor) run python run_dashboard.py, or the Colab notebook
        colab/Quantra_Barbershop.ipynb (Cell 1 → Cell 8). That's the X-ray tool; this is the cockpit.
